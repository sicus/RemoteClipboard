RemoteClipboard by Markus Biggel

Share your clipboard with other computers

With the RemoteClipboard you can share your Clipboard to other computers. Just connect to all RemoteClipboards you want to share your clipboard to, you can see the others clipboards too. You can set the clipboards to private state, so they are not send to the other clients or to public state to share them. Connections to multiple computers are possible. Only a TCP connection is required.

RemoteClipboard works on Windows and on Linux Computers, is a client running on Linux, the selection clipboard can also be shared.

To Build from Sourcecode just run:
qmake -makefile
make

You need the Qt 4.5 Library installed, for Windows you also need MinGW32 with g++ and make

You can also clone the project with Git by running: 

$ git clone git://github.com/sicus/RemoteClipboard
